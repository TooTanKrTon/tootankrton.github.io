Politique de Confidentialité

Dernière mise à jour : Mai 2026

Cette politique décrit quelles données sont collectées par les bots EAFC-GM et EAFC-ADMIN, comment elles sont utilisées et stockées, et quels sont vos droits.

1. Données collectées

Les bots collectent uniquement les données nécessaires à leur fonctionnement :

Donnée Source Utilisation
Identifiant Discord (ID utilisateur) Discord Identifier l'utilisateur dans le système
Pseudo Discord (display name) Discord Affichage dans les présentations et logs
Gamertag / Pseudo en jeu Saisi par l'utilisateur Fiche de présentation du joueur
Plateforme de jeu Saisi par l'utilisateur Fiche de présentation du joueur
Date de naissance / Âge Saisi par l'utilisateur Fiche de présentation du joueur
Poste(s) de jeu Saisi par l'utilisateur Fiche de présentation du joueur
Disponibilités Saisi par l'utilisateur Gestion des présences et convocations
Expérience / Compétitions Saisi par l'utilisateur Fiche de présentation du joueur
État d'avancement de l'inscription Bot (interne) Suivi du processus d'onboarding
ID et nom du serveur Discord Discord Isolation des données par serveur
✅ Les bots EAFC ne collectent aucune donnée financière, aucun mot de passe, aucune adresse email et n'accèdent pas aux messages privés des utilisateurs.

2. Stockage des données

  • Les données sont stockées sur les serveurs de l'hébergeur Discloud sous forme de fichiers JSON.
  • Les données sont isolées par serveur Discord : un serveur ne peut pas accéder aux données d'un autre.
  • Aucune base de données externe ni service tiers n'est utilisé pour stocker vos données.

3. Durée de conservation

  • Les données d'un utilisateur sont supprimées automatiquement lorsqu'il quitte le serveur Discord.
  • Les données d'un serveur sont supprimées si le bot est retiré du serveur.
  • Aucune donnée n'est conservée au-delà de la durée d'utilisation active.

4. Partage des données

Les données collectées ne sont jamais vendues, partagées ou transmises à des tiers. Elles sont utilisées exclusivement pour le fonctionnement des bots au sein de votre serveur Discord.

5. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : vous pouvez demander quelles données vous concernent.
  • Droit de rectification : vous pouvez modifier vos données via les commandes du bot (ex : /presentation).
  • Droit à l'effacement : vos données sont supprimées automatiquement en quittant le serveur, ou sur demande via le serveur officiel.

6. Contact

Pour toute question relative à vos données personnelles, vous pouvez nous contacter de deux façons :

  • Contacter directement le créateur sur Discord : tootankrton
